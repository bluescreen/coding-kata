# Coding Kata

Project for studiying Coding Kata and TDD in Typescript

Kata:

- fizzBuzz
  https://ccd-school.de/coding-dojo/function-katas/fizzbuzz/

Install

```
yarn install
```

Run the main code

```
yarn run
```

Run the tests

```
yarn test
```

Requires installed typescript and ts-node to run

```
npm -g install typescript ts-node
```
